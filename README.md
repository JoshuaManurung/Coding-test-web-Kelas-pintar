## Hasil Soal test Web

### Menjalankan Aplikasi
1. Download File dari Github
2. Kemudian jalankan XAMPP (Apache dan MySQL)
3. Kemudian buka CMD
4. cd ke Nama File yang sudah didownload
5. Jalankan php artisan serve
6. Buka localhost Anda
